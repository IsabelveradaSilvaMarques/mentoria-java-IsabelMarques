# Semana 1 – Fundamentos de Programação e Git/GitHub

Nesta semana foram estudados os conceitos básicos da linguagem Java, incluindo:
- Estrutura de um programa Java  
- Declaração e manipulação de variáveis  
- Estruturas condicionais (`if`, `else`, `switch`)  
- Estruturas de repetição (`for`, `while`, `do-while`)  
- Funções e métodos simples  

Também foi feita a configuração do ambiente (VS Code + Java 17) e a criação do repositório GitHub para versionamento de código.

## Estrutura desta pasta
- **exercicios/** → códigos desenvolvidos.
- **relatorio_diario/** → registos de progresso diário.
- **prints/** → imagens das atividades feitas.
